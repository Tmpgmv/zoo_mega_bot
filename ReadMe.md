# Телеграм-бот московсого зоопарка.
Для популяризации программы опеки над животными.

Бот-викторина помогает пользователю подобрать животное.

Бот позволит, с одной стороны — создать дополнительный формат 
развлечения для Telegram-канала зоопарка, а с другой — 
рассказать посетителям канала о программе опеки. 
Зоопарк надеется привлечь внимание взрослых работающих людей, 
которые любят животных и захотят стать опекунами.

# Запуск проекта
1. Клонировать проект.
```bash
git clone git@github.com:Tmpgmv/zoo_mega_bot.git
```

2. Установить зависимости
```bash
pip install -r requirements.txt
```

3. Применить миграции
```bash
python manage.py migrate
```
4. Создать администратора

```bash
python manage.py createsuperuser
```

5. Заполнить запросы и ответы викторины
```bash
python manage.py init_quiz
``` 

# Как пользоваться
1. Меню доступно по адресу /admin
Обязательно заполнить исходные данные:
   1) "Опции" (адрес веб-хука, URL для перехода в бот).
   2) О зоопарке (включая контактную информацию).
   3) Программа опеки.

2. Для администратора доступна страница /help
Здесь команды для смены веб-хука бота. А также ссылка на бота (URL для перехода в бот).

3. Мониторинг производительности доступен по адресу /metrics

4. Лог ошибок
Файл django_app.log в корневом каталоге проекта.

5. Контактный механизм
Связь с сотрудником зоопарка - через панель администратора.
Иначе говоря, если пользователь бота обращается
по поводу программы опеки, обращение появится
в панели администратора в разделе "Потенциальные опекуны".

6. Отзывы пользователей
В панели администратора в разделе "Отзывы"

# Масштабируемость
Результаты обращения к базе данных кешированы django-cachalot.
Т.е. на чтение из базы данных нагрузка может быть 
очень большой - тысячи обращений в секунду.

Запись в базу данных:  проектная нагрузка - до 1 изменения в секунду
(отзывы и обращения по вопросу опеки).

Если будет недостаточно, переходить на другие решения 
для кеширования (Memcached, Redis)

# Мониторинг производительности

Использован django-prometheus. 
Он собирает метрики в формате Prometheus, которые можно визуализировать в Grafana.

Эндпоинт с метриками: /metrics

Ключевые метрики для бота:

django_http_requests_latency_seconds — время ответа

django_db_query_duration_seconds — время SQL-запросов

django_db_execute_total — количество SQL-запросов
